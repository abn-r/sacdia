Asociación General

Arte en masa de pan

1. Hacer una lista de los materiales y los enseres que se necesitan para el arte en masa de 

pan.

2. ¿Por qué es importante tener las manos limpias y usar loción para las manos cuando se 

trabaja con masa de pan?

3. Explicar los diferentes procedimientos que se emplean para teñir y pintar la masa.

4. ¿Qué clase de acabado se usa para proteger y dar una apariencia brillante a la masa de 

pan?

5. Realizar por lo menos 2 diferentes flores con hojas como la rosa, la margarita, el narciso, 

el cornejo, la violeta, el jacinto de uva, el clavel, el lirio, el pensamiento, la amapola, la 

flor de pascua, el tulipán y la nomeolvides (se puede usar cortadores de masa). Arréglelos 

usando recipientes de su elección. (algunas sugerencias: canastas pequeñas, conchas, olla, 

etc.)

6. Realizar uno de los siguientes elementos usando masa teñida:

a. Un arreglo tipo escultura

b. Una pequeña placa para la pared

7. Realizar una de las siguientes tareas:

a. Un adorno con imán para el refrigerador

b. Un prendedor

c. Escoja algo que le gustaría hacer
