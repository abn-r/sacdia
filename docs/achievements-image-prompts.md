# Prompts para Generar Insignias de Logros

Cada prompt está listo para copiar y pegar en ChatGPT (DALL-E 3). Adjuntá siempre la imagen de `achievement-first-step-youversion-style.png` como referencia de estilo con el mensaje: *"Match exactly the style of this reference image."*

## Base del Estilo (usada en todos)

Todos los prompts mantienen:
- Flat matte design, NO glossy, NO 3D, NO reflections
- Engraved medallion / ancient coin aesthetic
- Thin geometric line art with uniform stroke weight
- Dark matte circular background (#1C1C1E)
- Thin circular border ring in the same accent color
- Square canvas 512x512, transparent outside the circle

---

## Archivo de salida sugerido

Guardá cada imagen como:
```
assets/generated/achievement-{slug}.png
```

# Tips para usar con ChatGPT

1. **Siempre adjuntá la imagen de referencia** (`achievement-first-step-youversion-style.png`) con cada prompt
2. Si la IA se desvía del estilo, respondé: *"Too glossy / too 3D. Make it completely flat like the reference image. Engraved coin style only."*
3. Generá todas en la misma conversación para mantener consistencia de estilo
4. Si una imagen queda rara, pedí: *"Regenerate with the exact same style as the reference, keep the engraved flat aesthetic."*
